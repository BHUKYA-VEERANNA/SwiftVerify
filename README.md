This project introduces a cutting-edge Multi-Factor Attendance System to meet the growing demand. Utilizing barcode scanning, facial recognition, gesture analysis, and voice matching, the system ensures robust user verification. Participants undergo swift barcode scans, followed by advanced facial recognition, gesture analysis, and voice matching for heightened security. The centralized database efficiently manages user information, updating attendance in real-time to a backend Excel sheet, minimizing manual errors. A standout feature is the automatic hourly report generation, providing valuable insights into attendance trends for educational institutions' technological advancement.
